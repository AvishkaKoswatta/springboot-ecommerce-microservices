# Spring Boot E-Commerce Microservices

A full-stack e-commerce application built using Spring Boot microservices architecture, React frontend, Docker, RabbitMQ, MySQL, and API Gateway.

## Tech Stack

### Backend

* Spring Boot
* Eureka Service Discovery
* RabbitMQ
* MySQL


### Frontend

* React

### DevOps

* Docker

---

## Microservices

* User Service
* Product Service
* Order Service
* Payment Service
* API Gateway
* Eureka Server

---

## Features

* User Authentication & Authorization
* JWT Security
* Product and Product Category Management
* Shopping Cart
* Order Management
* Payment Processing
* Service Discovery
* API Gateway Routing
* RabbitMQ Event Communication
* Dockerized Deployment
